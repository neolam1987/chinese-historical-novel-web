# 🎨 Anime Illustration Prompt Pack — Three Kingdoms Adventures

The story pages already have 20 hidden picture slots. As soon as you save each image
with the **exact filename** below into the folder **`images\anime\`**, it appears on
the page automatically (with its caption). No slot = nothing shown, so the site looks
fine even half-finished.

## How to use (about 10 minutes)

1. Open **Gemini** and pick the image model (Nano Banana).
2. Work on **one story at a time, in one chat** — after the first image, start the
   next prompt with *"Same characters and same art style as the previous image."*
   This keeps faces and costumes consistent.
3. Copy-paste one prompt per image. Ask for **landscape / 16:9** if it comes out square.
4. Download and save into `images\anime\` with the exact filename shown
   (keep the name even if Gemini gives you a .jpg — just rename the whole thing,
   e.g. `straw-1.png`; browsers don't mind).
5. Refresh the page — done!

**Style note baked into every prompt:** cute Japanese anime / Ghibli-style children's
storybook, watercolor, no blood, no text. If a result looks off, just reply "cuter,
softer, more storybook-like" and regenerate.
